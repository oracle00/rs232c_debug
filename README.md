RS232Cの通信実験用のPythonスクリプト
データはエンコードされない仕様
それぞれのスクリプト内でポートの指定を直接編集すること

serial_send.py 送信のみに対応。シェル上で送信したいデータを張り付けてエンターを押すと送信する。
serial_receive.py　受信のみに対応。受信が始まってから次の受信がタイムアウトするまでのデータをバッファリングしてから一度に出力する。
serial_transfer.py　２ポート間で、ポートAで受信したデータをポートBに、ポートBで受信したデータをポートAにパススルー転送する。通信履歴をテキストファイルに保存する。
serial_transfer_buff.py　２ポート間のデータをタイムアウト期間までバッファリングしてから転送する。通信履歴をテキストファイルに保存する。
